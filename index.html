📖 FAQ — Как это работает?

🎮 Для стримера

  • Монетизация: зрители донатят → получают билеты → участвуют в рулетке
  • Интерактив: каждый раунд — шоу с анимацией и напряжением
  • Автоматизация: бонусы, джекпоты, события происходят сами (настраиваемые шансы)
  • Удержание: система баллов и штрафов мотивирует возвращаться
  • Гибкость: все настройки под себя (проценты, режимы, правила)

🎯 Для игроков

  • Шанс для всех: даже с 1 билетом можно выиграть (инверсия шансов)
  • Прогресс: баллы копятся → можно конвертировать в билеты (5 баллов = 1 билет)
  • Нет потери: победитель получает все билеты обратно в баллы (1:1)
  • Бонусы: авто-события, джекпоты, меценаты — дополнительные шансы
  • Честность: настоящий рандом, всё прозрачно, результаты видны

⚙️ Как пользоваться

  1. Зритель донатит → вводим ник, сумму, тип (игрок/болельщик/меценат)
  2. Набираем 5+ участников с билетами
  3. Жмём F1 или кнопку "ИГРАТЬ" → рулетка крутится
  4. Победитель получает билеты → баллы, проигравшие -1 билет +1 балл
  5. Авто-события срабатывают сами (настраивается в %)
  6. Новый стрим → адаптация (сброс билетов, сохранение баллов)

🔧 Что настраивается

  • Авто-событие кубика: шанс %, билеты (1-3) или баллы всем
  • Авто-джекпот: шанс %, половина билетов ИЛИ баллы всем
  • Авто-бонус: шанс %, 1-5 баллов случайному
  • Инверсия: для бонусов (меценат/джокер) — меньше билетов = больше шанс
  • Штрафы: вкл/выкл, -1 билет +1 балл проигравшим
  • Удаление: победитель удаляется из круга (конвертация в баллы)

💡 Почему так сделано?

Билеты → Баллы: победитель не теряет прогресс, может вернуться в следующем раунде
Инверсия: даёт шанс тем, кто мало задонатил — интереснее для всех
Авто-события: создают динамику, не дают заскучать между раундами
Штрафы + Баллы: мотивируют играть аккуратно, но дают второй шанс
Меценаты: большие донаты запускают цепочку бонусов для всех
Адаптация: каждый стрим — новая история, но прогресс (баллы) сохраняется

💾 Все данные сохраняются в localStorage (браузер). Экспорт/импорт для резервных копий.

️ При очистке браузера данные удалятся — используйте экспорт!
